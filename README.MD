Inline math equations go in like so: $\omega = d\phi / dt$. 
Displaymath can be written as:

\begin{equation}
I = \int \rho R^{2} dV
\end{equation}

You can backslash-escape any punctuation characters
which you wish to be displayed literally, ex.: \`foo\`, \*bar\*, etc.
